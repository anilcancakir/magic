# Magic

A new magical flutter package.

## Features

- Application container
- Environments
- Auth
- API base HTTP
- Model
- Controller
- Translation
- And much more...

## Getting started

Clone the repository

```
git clone https://github.com/anilcancakir/magic.git
```

Go to path

```
cd magic
```

Run the flutter

```
flutter run
```
